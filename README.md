# NovaSoft - Premium Startup Website Template

A modern, responsive, and high-performance website template for a software development company.

## Features
- **Modern UI/UX**: SaaS-inspired aesthetic with premium gradients and typography.
- **Light/Dark Mode**: Integrated theme toggle with persistence.
- **Responsive Design**: Fully optimized for Mobile, Tablet, and Desktop.
- **Micro-animations**: Smooth scroll reveals and hover effects.
- **Interactive Components**: Sticky navbar, mobile hamburger menu, and smooth scrolling.
- **Working Contact Form**: Validated mock submission logic.
- **SEO Ready**: Semantic HTML5 structure.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (ES6+)
- Font Awesome (Icons)
- Google Fonts (Inter & Outfit)

## Folder Structure
```text
startup-website/

├── js/

```

## How to Run Locally
1. **Download/Extract** the project folder.
2. **Open index.html** in any modern web browser (Chrome, Firefox, Safari, Edge).
3. **Optional (Live Server)**: If using VS Code, right-click `index.html` and select "Open with Live Server" for the best experience.

## Theme Switching
The website supports both light and dark modes. Click the **moon/sun icon** in the navigation bar to switch. Your preference is automatically saved to your browser's local storage.
